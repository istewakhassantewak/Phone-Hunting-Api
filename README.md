📱 Phone Hunting API

🔗 Live Site: https://phone-hunting-api-ruddy.vercel.app

A responsive phone search application that allows users to search and explore smartphone information dynamically using an external API. The project focuses on API integration, dynamic data rendering, and interactive frontend functionality.

🚀 Features
🔍 Search smartphones dynamically
📡 Real-time API data fetching
📱 Detailed phone information display
⚡ Fast and interactive user experience
🎨 Clean and responsive UI design
📂 Dynamic content rendering with JavaScript
📱 Mobile-friendly responsive layout

🛠️ Technologies Used
HTML5
Tailwind CSS
JavaScript (Vanilla JS)
REST API

📁 Project Structure
├── js/                   # JavaScript files
├── index.html            # Main HTML file
├── tailwind.config.js    # Tailwind configuration
├── README.md             # Project documentation
└── .gitattributes

⚙️ Setup & Usage
Clone the repository:
git clone https://github.com/istewakhassantewak/Phone-Hunting-Api.git
Navigate to the project folder:
cd Phone-Hunting-Api
Open index.html in your browser.

🔎 Core Functionalities
Search phones by keyword
Fetch phone data from API
Display phone cards dynamically
View detailed device information
Responsive search interface
